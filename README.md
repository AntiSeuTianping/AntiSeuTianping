# AntiSeuTianping
该项目灵感由[996.ICU](https://github.com/996icu/996.ICU)启发，用于收集东南大学九龙湖校区桃园天平超市经营中的各种不规范行为。
## 我能做什么?
+ 在`Evidence`文件夹下上传你所拥有的凭证，具体步骤请看`Evidence`文件夹下的`README.MD`文件。
+ 如果你对如何维权有较为深刻的认识(如法学相关专业的同学), 你可以选择整理这些凭证并通过你的维权渠道进行维权。
+ 给该项目提供新的发展建议和思路。
+ 对于天平超市定价不公的商品，拒绝购买。
+ 扩展该Repo的影响
